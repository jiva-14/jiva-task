#  Task 4: CSS - Units, Box Models, Font

##  Description
A simple webpage built with HTML & CSS that demonstrates:
- CSS Units (rem, vh, vw, em)
- Box Model (margin, padding, box-sizing)
- Font Styling (size, weight, spacing)
- Gradient Button
- Viewport based layout with no scroll

##  Tech Used
- HTML5
- CSS3

## 🎯 CSS Concepts Used

| Property        | Value        | Purpose                  |
|----------------|--------------|--------------------------|
| margin          | 25% / 10vh   | Spacing left/right/top   |
| box-sizing      | border-box   | Prevent overflow         |
| font-size       | 1.2rem       | Button text size         |
| letter-spacing  | 0.1em        | Space between letters    |
| text-transform  | uppercase    | Capital letters          |
| background      | linear-gradient | Gradient button       |
| height          | 80vh         | No scroll on page        |

## Expected Output
- Image centered with 25% space on each side
- 10% space from top and bottom
- Gradient yellow-to-green button below image
- No scrollbar visible
